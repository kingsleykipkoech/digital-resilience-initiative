# digital-resilience-initiative
Digital Resilience Initiative by Kingsley Kipkoech

This repository contains the static website portfolio for the final submission of Kingsleykoech's Venture Creation project.

## Project Summary
The project focuses on developing an **Accessible, Open-Source Security Platform** coupled with **Integrated Digital Literacy Training** to combat the systemic cybersecurity vulnerability faced by Small Businesses and Non-Profit Organizations (SBNPOs) in developing regions (Nairobi, Kenya).
